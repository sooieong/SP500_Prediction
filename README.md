# 개요
최근 3년간의 S\&P 500 시장 데이터를 기반으로 최적의 예측 방법론을 도출하기 위해 두 가지 데이터 구성 전략과 다양한 모델링 기법을 체계적으로 비교 및 검증합니다. 목표는 데이터 구성과 모델링 기법 중 어느 조합이 가장 우수한 예측 성능(RMSE, R^2)을 보이는지 확인하는 것입니다.

## 데이터 구성 전략 비교
- **버전1(top60)**: S&P 500 시장 전체의 움직임을 '대표적인 소수 기업'의 평균적인 움직임으로 대체하여 분석하는 전략입니다. S&P 500 지수에 가장 큰 영향력을 미치는 60개 기업의 개별 주식 데이터와 투자 지표(per, pbr 포함 여러개)의 평균값을 사용하였으며, 보조 데이터로 환경 변수를 추가합니다.

- **버전2(SPY)**: SPY ETF는 S&P 500의 500개 종목 전체를 담고 있으므로 SPY의 데이터 자체가 가장 포괄적인 시장의 성과를 나타냅니다. SPY ETF의 주식 데이터 및 투자지표(PER, PBR)를 사용하였으며, 보조 데이터로 환경 변수를 사용합니다.

## 데이터 전처리
버전1(top60)과 버전2(SPY) 데이터 셋에 동일한 데이터 전처리 과정을 적용하여 모델 학습의 안정성과 신뢰도를 높입니다.
- **로그 변환 (Log Transformation)**: 데이터의 왜곡된 분포를 정규 분포에 가깝게 만들어 모델의 가정 충족 및 이상치의 영향을 완화합니다.
- **다중공선성 변수 제거 (Collinearity Removal)**: 독립 변수들 사이에 높은 상관관계가 있는 경우 해석 안정성을 해치거나 과적합을 유발할 수 있습니다. 이를 사전에 제거하여 모델의 신뢰성을 높입니다.
- **차분 (Differencing)**: 대부분의 금융 시계열 데이터(주가 자체)는 비정상성의 특징이 있지만, 차분을 통해 정상성을 띠게 만들어 모델이 추세보다는 패턴과 변동성을 학습하도록 돕습니다.

## 데이터 모델링
전처리가 완료된 버전1(top60)과 버전2(SPY) 데이터 셋에 **Multi Regression, Light GBM, XGBoost, LSTM, GRU** 등의 모델링을 진행합니다.

모델링 결과는 **RMSE, R2** 등의 성능 평가 지표를 통해 비교 및 검증합니다.
